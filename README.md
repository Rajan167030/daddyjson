## Getting Started

### Prerequisites

- Node.js (vXX or higher)
- npm or yarn

### Installation

Clone the repository:

```bash
git clone https://github.com/ken-rolex/daddy-json.git
cd daddy-json
Install dependencies:

bash
npm install
# or
yarn install
Running the Project
To start the development server:

bash
npm start
# or
yarn start
Folder Structure
src/: Main TypeScript source code
public/: Static assets
styles/: CSS files
